# CCB Rev C Task Tracker

Private progress tracker for 75809910 CCB Rev C board spin (HWD-12800).

## View
Open the GitHub Pages URL shared with you.

## Publish Progress (Eduard only)
1. Open the tracker in your browser
2. Check off completed tasks
3. Click **Publish Progress** button (bottom bar)
4. Run `.\publish.ps1` in PowerShell from this folder
